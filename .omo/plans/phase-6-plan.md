## 2. Phase 6 目标 (6 个月)

### 2.1 商业目标
- 客户: 10 → 200 (月增长 25-30%)
- ARR: ¥0 → ¥5M (¥299 × 1500 客户 × 平均 1.2 seat)
- 团队: 5 → 25 人 (销售 8 / 工程 12 / 运营 5)
- 资金: 启动种子轮 ¥3-5M (现有 1 客户白鼠成功故事)

### 2.2 技术目标
- 多区部署: 北京 (主) + 上海 (灾备) + 广州 (CDN)
- 性能: P99 < 200ms, 错误率 < 0.1%
- 等保三级 + ISO 27001 + SOC 2 Type II
- AI 准确率: 简历解析 95% → 99%, 评估相关性 +20%
- 数据看板: 实时 (现 T+1)
- 集成: 10+ (钉钉/飞书/企微 + 招聘平台 Boss直聘/拉勾/猎聘 + HRIS)

## 3. Phase 6 任务分解 (P7-1 ~ P7-20)

### 3.1 规模化基建 (P7-1 ~ P7-6, 估 30d)
- P7-1 多区部署 + 灾备演练 (北京主 / 上海备)
- P7-2 Kubernetes 迁移 (现 Docker Compose)
- P7-3 Redis Cluster + PostgreSQL 主从
- P7-4 CDN 智能调度 (国内多 CDN: 阿里云/腾讯云/Cloudflare 中国)
- P7-5 等保三级 (三级备案 + 测评 + 整改)
- P7-6 ISO 27001 (信息安全管理体系认证)

### 3.2 AI 能力增强 (P7-7 ~ P7-10, 估 30d)
- P7-7 AI 准确率提升 (微调 + RLHF, 简历解析 99%)
- P7-8 多模态评估 (视频面试分析: 微表情/语速/亲和力)
- P7-9 行业垂直模型 (互联网 / 金融 / 医疗 / 制造 4 套)
- P7-10 AI 自学习 (客户 override → 训练数据, 持续提升)

### 3.3 销售/运营工具 (P7-11 ~ P7-15, 估 30d)
- P7-11 销售 pipeline CRM (对接 Salesforce/HubSpot)
- P7-12 客户成功平台 (Gainsight 替代, 自研)
- P7-13 Marketing 自动化 (邮件/微信群发, 行为触发)
- P7-14 计费平台 (订阅/续约/发票, 自动化)
- P7-15 客户教育平台 (Academy + 认证 + NPS 调研)

### 3.4 集成生态 (P7-16 ~ P7-20, 估 30d)
- P7-16 招聘平台集成 (Boss直聘/拉勾/猎聘/智联)
- P7-17 HRIS 集成 (薪人薪事/北森/SAP SuccessFactors)
- P7-18 SSO + SCIM (Okta/Azure AD/钉钉/飞书/企微)
- P7-19 Zapier 风格自动化 (webhook + 工作流)
- P7-20 公开 API + 开发者中心 (docs.airecruit.com)

## 7. Phase 6 决策点 (D180 复盘)

如果 D180 数据:
- 客户 < 5: **暂停扩招, 优化产品**
- 客户 5-15: **继续, 目标 D360 30 客户**
- 客户 15+: **加速, 启动 A 轮 (¥10-20M)**
- 客户 30+: **进入 Phase 7 (国际化 or 行业垂直)**
