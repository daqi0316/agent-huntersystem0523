# AI 招聘助手 — 帮助中心 (FAQ)

## 产品使用

### 简历导入
**Q: 支持哪些简历格式?**
A: PDF / Word (.docx) / 图片 (jpg/png) / 纯文本粘贴。

**Q: 单次最多导入多少份?**
A: 单组织 100 份/批, 月度 1 万份。超限请 [联系升级](mailto:sales@airecruit.com)。

**Q: 候选人重复怎么办?**
A: 系统按手机号/邮箱自动去重, 重复会提示合并。

### AI 评估
**Q: AI 评估依据是什么?**
A: 详见 [AI 评分披露](../legal/privacy-policy.md#8-ai-评估特别说明)。简言之: 评估依据 + 关键能力维度 + 置信度均向 HR 公开, 不可识别性别/民族/年龄。

**Q: 候选人对 AI 评分有异议怎么办?**
A: 候选人 7 天内可申诉, 我们 48h 内人工复核。详见 [P5-10 AI 监管](https://github.com)。

### 多组织 / 权限
**Q: 怎么邀请同事?**
A: 组织管理员 → 设置 → 成员 → 邀请 (微信扫码或链接)。邀请可指定角色: Owner / Admin / HR / Interviewer / Viewer。

**Q: 老带新怎么用?**
A: 在 [推荐页面](https://app.airecruit.com/referral) 生成推荐码, 双方各获 1 个额外 seat。

## 计费 / 退款

### 试用
**Q: 试用是免费的吗?**
A: 14 天全功能免费, 无需信用卡。试用结束前 3 天提醒, 降级不扣款。

**Q: 试用能用 AI 评估吗?**
A: 14 天试用 = 完整付费功能, 含 AI 评估 100 次。

### 订阅
**Q: 价格多少?**
A: 见 [价格页](https://airecruit.com/pricing)。SMB ¥299/月起, Enterprise ¥1999/月起, 定制请询销售。

**Q: 可以月付/年付切换吗?**
A: 随时切换。年付 8 折 (节省 2 个月)。

### 退款
**Q: 不满意能退吗?**
A: 7 天无理由退款 (订阅周期内)。详见 [退款 SOP](../refund-dispute-sop.md)。

**Q: 怎么申请退款?**
A: [提交工单](https://app.airecruit.com/support) 选择"退款申请", 5 个工作日内原路退回。

## 数据 / 合规

**Q: 我的数据存在哪?**

**Q: 数据怎么备份?**
A: 日备 + 异地灾备 (RPO 1h, RTO 4h), 备份加密。
