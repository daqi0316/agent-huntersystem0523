## 5. 服务内容

### 5.1 网站名称
- 中文名称: 智聘招聘助手
- 英文名称: AI Recruitment Assistant (不做主名称)

### 5.2 服务类型
- 互联网信息服务 — 非经营性

### 5.3 网站内容说明
AI 招聘助手是 B2B SaaS, 为企业 HR 提供:
1. 简历智能解析 (候选人授权数据)
2. AI 评估候选人能力 (多维度)
3. 面试安排与管理
4. 团队协作 + 权限管理
5. 数据看板 + 增长分析

- 经营性互联网信息服务 (无广告/电商/付费推广)
- 互联网新闻信息 / 出版 / 文化 / 视听节目
- 金融 / 医疗 / 教育特殊行业

### 5.4 行业类别
- 行业大类: 信息技术
- 行业小类: 软件开发 / 信息技术服务

- 示例:
  ```
  京 ICP 备 2026000000 号-1  ·  京公网安备 11010500000000 号
  ```

### 7.2 备案信息公示
- 主体信息变更: 30 天内备案变更
- 域名 / IP 变更: 提前 5 个工作日变更
- 接入商变更: 与新接入商重签接入协议

### 7.3 年度检查
- 每年 3-6 月 期间, 接入商发起"年度备案信息准确性核查"
- 需在 5 个工作日内回复, 确认主体/网站/负责人信息无变更

### 7.4 公安备案 (上线后 30 天内)
- 登录 http://www.beian.gov.cn 提交
- 主体同, 但"网站"维度
- 审核周期 7-30 天


## 10. 风险与备选

### 10.1 备案被驳回常见原因
| 原因 | 解决方案 |
|---|---|
| 经营范围不含"互联网信息服务" | 变更营业执照经营范围 (3-5 工作日) |
| 域名未实名认证 | 阿里云万网提交实名认证 (1-3 工作日) |
| 主体负责人与网站负责人核验失败 | 重新拍照, 保持表情自然、背景干净 |
| 接入 IP 变更频繁 | 申请"阿里云备案专用 IP", 提交后 7 天不变更 |
| 网站名称敏感/违反规范 | 改名 (如去"智聘" → "智选聘") |

### 10.2 备案前网站已上线风险
- 工信部发现后, **注销备案 + 罚款 1-10 万 + 关停网站**
- **必须先备案, 再上线**

### 10.3 应急方案
- 备案期间用 `airecruit.com` 暂时 301 到宣传页 (无功能)
- 备案号下来后再开功能
- 客户白鼠 (内部测试账号) 不上线 = 无需备案
