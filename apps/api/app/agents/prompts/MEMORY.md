## 1. 面试轮次标准（4 轮制）

| 轮次 | 类型 | 时长 | 面试官 | 重点评估 |
|------|------|------|--------|---------|
| R1 | 电话初筛 | 30 min | HR 专员 | 基础背景、意向确认、硬性条件验证 |
| R2 | 技术面 | 60 min | 直属 Leader + 资深同事 | 技术深度、编码能力、项目经验 |
| R3 | 行为 / 系统设计 | 60 min | 协作部门代表 | 软技能、架构能力、跨团队协作 |
| R4 | 终面 / 交叉面 | 45 min | 部门 Head / VP | 文化适配、价值观、战略思维 |

**特殊场景**：
- 应届生 / 校招：R1 改为 20 min 视频，R2 改为在线笔试 + 1 面合并
- 高管 / VP+：增加 R5 董事会面（30 min），R1 可跳过
- 跨地域：R1 远程，R2-R4 视情况混合

## 2. 反馈时效

- **24 小时硬性 SLA**：面试结束后 24h 内必须提交反馈
- **超时未提交**：
  - 第 24h：自动催办通知面试官
  - 第 48h：升级到面试官直属 Leader
  - 第 72h：升级到 HR 负责人，自动标"需补面"
- **反馈质量不达标**（缺维度分 / 缺关键观察）：HR 审核 24h 内退回重写

## 3. 一票否决项（5 条硬性）

| # | 类型 | 说明 |
|---|------|------|
| 1 | 硬性技能完全缺失 | JD 要求的核心技能，候选人 0 经验（如要求 Python，简历无任何编程经验）|
| 2 | 学历严重不达标 | JD 要求硕士，候选人本科（除非特批）|
| 3 | 经验年限严重不足 | JD 要求 5 年，候选人 < 1 年（除非降级）|
| 4 | 诚信风险 | 学历 / 工作经历 / 项目经验造假证据充分 |
| 5 | 法律合规风险 | 竞业限制未解除、签证问题、未达法定就业年龄 |

**触发一票否决** → 流程立即终止 + 标注 `rejected_by_veto` + 通知 HR + 不可被后续轮次"复活"。

## 4. 渠道优先级（按岗位类型）

| 岗位类型 | 优先渠道 | 备选渠道 | 慎用 |
|---|---|---|---|
| **高端技术**（P7+ 算法 / 架构） | LinkedIn + 猎头 + GitHub | 内部推荐 / 行业峰会 | 招聘平台 |
| **互联网产品 / 运营** | Boss 直聘 + 脉脉 | 智联 / LinkedIn | 猎头（成本）|
| **应届生 / 校招** | 校招 + 实习转正 | 牛客 / 实习僧 / 学校社群 | 招聘平台 |
| **传统行业**（制造 / 金融 / 医药）| 智联 + 前程无忧 | 现场招聘会 / 行业协会 | LinkedIn |
| **高管 / VP+** | 猎头网络 + 行业峰会 | 内部推荐 + 董事人脉 | 公开招聘平台 |

**渠道 ROI 监控**：每季度 review 一次各渠道的"申请→offer→入职"转化率，成本过高的渠道降级或弃用。

## 5. 薪酬带宽（2026 Q2，研发序列）

> 单位：人民币 / 月，base salary。**年度总包 = base × 14-16 + 股票 + 福利**。

| 级别 | 经验 | Base 范围（月） | Total 范围（年） | 备注 |
|------|------|----------------|----------------|------|
| P5（初级）| 0-3 年 | 15-25K | 18-30 万 | 应届 / 1-2 年 |
| P6（中级）| 3-5 年 | 25-40K | 30-50 万 | 独立 owner 小模块 |
| P7（高级）| 5-8 年 | 40-65K | 50-80 万 | 跨团队 owner |
| P8（资深）| 8-12 年 | 65-100K | 80-120 万 | 团队 leader |
| P9（专家）| 12+ 年 | 100-150K | 120-200 万 | 跨 BU 影响力 |

**特殊调整**：
- **稀缺技能溢价**（AI / 区块链 / 芯片 / 量化）：+20-30%
- **跨城市溢价**（北京 / 上海 / 深圳）：+10-15%
- **回流溢价**（从海外回流）：+15-25%
- **特批通道**：超出 P9 带宽需 CTO 审批，无上限

**保密原则**：薪酬带宽仅 HR 负责人 + 部门 Head 可见，**不向候选人展示具体数字**。

## 7. 风险信号清单（简历筛选用）

- **gap**：简历中 >3 个月的空窗期未说明
- **job_hopping**：每份工作 <1 年
- **skill_inflation**：技能描述可能夸大（如"精通" + 缺乏深度证据）
- **salary_mismatch**：期望薪资偏离市场水平 >50%
- **integrity_risk**：教育 / 工作经历可能不实
- **competition_risk**：未确认竞业限制
- **motivation_mismatch**：求职动机与公司方向不匹配

**触发 ≥2 条 high 风险** → 自动标"需人工复核"，不直接拒。

## 8. 试用期规范

- **30-60-90 天 review 节点**：
  - 30 天：环境适应 + 基础任务
  - 60 天：独立承接项目
  - 90 天：转正评估
- **不通过处理**：30 天内不通过 → 协商离职；60/90 天不通过 → 延长试用期或协商离职
- **HR 跟进频率**：每 2 周 1 次 1-on-1，记录到 on-track / at-risk / off-track
