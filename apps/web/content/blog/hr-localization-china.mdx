# 海外 HR 进入中国市场的 5 个本地化坑

越来越多海外 HR SaaS (Greenhouse / Lever / Workable) 想进中国, 但本地化复杂, 90% 会在 2 年内退出。

本文拆解我们作为本土厂商, 看到的 5 个最常见坑 + 解决方案。

## 坑 1: 忽视 ICP 备案, 上线即被叫停

### 1.1 现象
海外厂商常以为"中国用户用海外 SaaS 没问题" — 但工信部要求**所有面向中国用户的网站**必须 ICP 备案, 否则罚款 1-10 万 + 关停。

### 1.2 真实案例
某海外 ATS 厂商 2024 年被工信部下架, 损失 200+ 中国客户。

### 1.3 解决方案
- 注册中国境内公司 (WFOE / JV)
- 阿里云 / 腾讯云接入, 走 ICP 备案流程 (7-20 工作日)
- 公安备案 (上线 30 天内)
- 悬挂备案号 + 公安备案号

**耗时**: 30-60 天准备 + 7-20 天审核

## 坑 2: 强行用 Email 沟通, 中国用户不看邮件

### 2.1 现象
海外 SaaS 默认 Email 通知。中国 HR 一天收 100+ 邮件, 重要通知直接淹没。

### 2.2 真实数据
- 中国 HR 微信打开率: 95%+
- 中国 HR 邮件打开率: 8-12%

### 2.3 解决方案
- 主要通知走**微信服务号 + 模板消息**
- 次要走**短信** (阿里云 / 腾讯云短信)
- Email 仅作辅助 (法律通知 / 收据)
- 3 通道冗余, 至少 1 个送达

## 坑 3: 不懂个保法 (PIPL), 直接搬 GDPR 模板

### 3.1 现象
海外厂商常把 GDPR 隐私政策翻译成中文, 直接用。PIPL 比 GDPR 严, 多项要求无对应:

- **敏感个人信息**: 身份证/健康/生物识别 = 单独同意
- **跨境传输**: 需 PIPL 评估 + 标准合同
- **自动化决策**: AI 评估对个人权益有重大影响, 需提供说明
- **数据主体权利**: 7 个独立权利, 部分 GDPR 无对应

### 3.2 真实案例
某海外 ATS 因未提供 AI 评估说明, 被候选人起诉, 罚款 50 万 + 公开道歉。

### 3.3 解决方案
- 单独写 PIPL 隐私政策 (不是 GDPR 翻译)
- AI 评估 100% 透明: 依据 + 维度 + 置信度
- 人工 override + 审计链 (7 年留存)
- 候选人 7d 申诉 + 48h 人工复核

## 坑 4: 用海外 LLM (GPT/Claude), 触红线

### 4.1 现象
海外厂商默认接 OpenAI/Anthropic。中国监管要求**生成式 AI 服务需备案**, 跨境 LLM 调用需"安全评估"。

### 4.2 真实数据
2023 年起, 工信部 / 网信办多次发文规范生成式 AI:
- 《生成式人工智能服务管理暂行办法》(2023-08)
- 训练数据 + 模型需备案
- 输出内容需"安全可控"

### 4.3 解决方案
- AI 推理用国内 LLM: 通义千问 / 文心一言 / 智谱 GLM / DeepSeek
- 客户数据**不**出境 (用国内 LLM API)
- 不训练通用模型 (避免触发"基础模型"备案)
- 输出走内容安全中间件 (黄/政/暴/敏感词)

## 坑 5: 客服 9-5 美西时间, 中国用户上班时找不到人

### 5.1 现象
海外厂商默认支持美西时间 9-5 (中国 0-8 AM), 中国 HR 上班时客服睡觉。

### 5.2 用户反馈
"提交工单 2 天没人回" 是 N-1 流失原因。

### 5.3 解决方案
- 客服覆盖**中国工作时间**: 9:00-21:00 (含周末)
- 多渠道: 工单 / 微信 / 邮件 / 电话
- 1 工作日内响应, P0/P1 < 1h
- 海外总部 + 中国团队双班

## 总结: 海外 HR 厂商进中国 checklist

| 项 | 必做 | 耗时 | 成本 |
|---|---|---|---|
| 注册 WFOE | ✅ | 30-60d | 10-30 万 |
| ICP 备案 | ✅ | 7-20d | 自办 |
| 公安备案 | ✅ | 7-30d | 自办 |
| PIPL 隐私政策 | ✅ | 14-30d | 律师 5-10 万 |
| AI 透明度 + 申诉 | ✅ | 30-60d | 工程 20-30 万 |
| 国内 LLM 接入 | ✅ | 7-30d | 工程 5-10 万 |
| 微信集成 | ✅ | 30-60d | 工程 10-20 万 |
| 短信通道 | ✅ | 7d | 充值 1-3 万 |
| 客服本地化 | ✅ | 14-30d | 人力 30-50 万/年 |
| 等保三级 | ✅ | 90-180d | 测评 15-30 万 |
| **总计** | | **6-12 个月** | **150-300 万** |

**结论**: 海外 HR 厂商进中国 = 至少 1 年 + 200 万 RMB 投入, 还不算客户教育。

本土厂商优势: 6 个月上线 + 30 万成本, 这是 5-10x 差距。
